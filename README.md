# RobotPIDSimulator
A tool to simulate a robot moving to a target using Proportional Integral Derivative (PID) control.

## About the Project
* It is a way to smoothly control a robot's movement to its target position.
* It measures the current position of the robot, computes its distance from the target, and uses the PID to decide the robot's movement.
* I'm uploading it to Github for application evidence

## Why I built this:
This was a way for me to learn robotic fundamentals, such as motion and how code uses path to apply to movement in a robot.

## Development Timeline:
This project took about a week (around 5 days), and I worked on it over winter break. 

## Technologies Used:
* Java 25—The programming language used for this project
* VS Code - My code editor

# How to run:
* Copy files to a folder
* Run Main.java

## What I learned:
* Robots control velocity, and that position changes over time.
* Small corrections are used instead of one big movement for stability.
* The need for time-based simulations and constraints
